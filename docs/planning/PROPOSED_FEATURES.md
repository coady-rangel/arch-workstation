# Proposed Features

This document serves as a planning and brainstorming space for ideas that may improve the project.

Items in this document are **not committed work**.

Only features that have been evaluated and approved should be promoted to `docs/roadmap/ROADMAP.md`.

---

# Deployment

## Host Profiles

Support multiple workstation profiles.

Examples:

- Desktop
- Laptop
- Virtual Machine
- Development Workstation

---

## Optional Components

Allow users to selectively install components during deployment.

Examples:

- Gaming
- Development
- Virtualization
- Multimedia
- AI Workstation

---

## Plugin Architecture

Support modular feature packages that can be enabled or disabled independently.

---

## Interactive Installation

Provide an optional guided installation experience while preserving unattended deployment.
