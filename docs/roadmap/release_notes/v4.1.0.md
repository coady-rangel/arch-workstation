# Arch Workstation v4.1.0

## Overview

This release introduces the Engineering Toolkit, a modular interactive console for operating, validating, and supporting the Arch Workstation repository.

The Toolkit provides a foundation for future engineering, deployment, configuration, documentation, and operational workflows while continuing to use the repository's existing scripts as the source of truth.

---

## Added

- Top-level interactive Toolkit launcher
- Modular Toolkit architecture with dynamic module loading
- Shared Toolkit library
- Engineering module
- Support module
- Live Engineering dashboard
- Repository branch and upstream status reporting
- Ahead and behind commit reporting
- Staged, modified, and untracked file counts
- Shared script execution helper
- Shared confirmation, formatting, document-viewing, and command helpers
- Toolkit architecture documentation

---

## Engineering Workflows

The Engineering module provides access to:

- Repository status
- Change review
- Repository Doctor
- Repository validation
- Workstation validation
- Release Assistant

---

## Reliability Improvements

- Child scripts may return non-zero exit codes without terminating the Toolkit.
- Script results and exit codes are reported before returning to the menu.
- Shared behavior is centralized in `toolkit.d/lib/common.sh`.
- Existing repository scripts remain the canonical implementation for engineering operations.

---

## Foundation for Future Work

This release establishes the Toolkit as the operational interface for the Arch Workstation project.

Planned expansion areas include:

- Guided engineering workflows
- Deployment and repair operations
- Documentation utilities
- Configuration management
- System operations and health checks
