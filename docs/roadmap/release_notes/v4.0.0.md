# Release Notes

## Version

v4.0.0

## Release Date

2026-07-23

---

# Overview

This release introduces the initial Support Framework for the Arch Workstation project. The framework provides a modular architecture for collecting system diagnostics and generating support bundles suitable for troubleshooting and future expansion.

---

# New Features

## Support Framework

- Introduced modular collector architecture
- Added shared support library
- Added bundle output management
- Added latest bundle symlink support

## Collectors

Implemented the following collectors:

- System
- Hyprland
- Waybar

## Summary Generation

- Added automatic support bundle summary generation
- Reports collector health
- Reports overall bundle status

## Documentation

Added:

- Support framework documentation
- Collector contract
- Collector template
