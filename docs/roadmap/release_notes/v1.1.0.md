# Release Notes — v1.1.0

Release Date: 2026-07-19

---

## Summary

Version 1.1.0 modernized workstation session management by replacing the previous wlogout implementation with a Rofi-based power menu. This simplified the user experience while improving maintainability and customization.

---

## Highlights

- Replaced wlogout with a Rofi power menu.
- Added dedicated power menu launcher script.
- Updated Waybar integration.
- Added required package dependencies.

---

## Notable Changes

### Added

- Rofi-based power menu.
- Power menu launcher script.

### Improved

- Waybar integration.
- Power management workflow.

### Removed

- Dependency on wlogout for session management.

---

## Upgrade Notes

Users upgrading from previous releases should use the new Rofi power menu in place of the legacy wlogout interface.

---

## Looking Ahead

Future releases will continue refining the desktop environment and workstation automation.
